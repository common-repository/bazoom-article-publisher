<p> The Bazoom plugin is a tool used to publish articles on websites. To use this plugin, websites need to register with Bazoom.</p>

Important Note: This plugin requires PHP 7.2 or higher and WordPress 5.9.0 or higher. Please update your PHP and WordPress versions before using this plugin.

== Third-Party Service Integration: Bazoom ==

This plugin utilizes the Bazoom service (https://article-plugin.bazoom.net) to publish articles on your website. When activated, the plugin establishes a connection with Bazoom's API to perform tasks such as:

    * Retrieving available categories for article publishing
    * Sending article content and metadata for publication
    * Checking activation status and managing API keys

Data Usage and Privacy:

The plugin transmits the following data to Bazoom:

    * Article title, content, and metadata (e.g., categories, tags)
    * Website URL and basic information
    * Bazoom API key (for authentication)

Please note that Bazoom has its own privacy policy outlining data collection and usage practices: (https://bazoom.com/terms/cookie-privatlivspolitik). We recommend reviewing this policy to understand how Bazoom handles your data.

User Consent and Responsibility:

By using this plugin, you acknowledge that your website will be transmitting data to Bazoom as described above. We encourage you to inform your website users about this integration and its data implications.

Additional Notes:

* This plugin does not store any Bazoom-specific user data on your website.
* You can manage your Bazoom API key and other settings from the plugin's configuration page.
